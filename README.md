# Global Film Genre Dashboard

This project analyzes the distribution and popularity of film genres across countries using Excel.  
It includes an interactive dashboard with a heat map, ranking table, and bar chart controlled by a genre slicer.

## 🔍 Objectives
- Explore how film genres vary across countries
- Identify top countries for each genre
- Build an interactive Excel dashboard
- Practice data cleaning, pivot modeling, and visualization

## 📊 Dashboard Features
- **Genre Heat Map** showing distribution across countries
- **Top 20 Countries Table** ranked by selected genre
- **Bar Chart** visualizing country popularity
- **Genre Slicer** for interactive filtering

## 🛠 Tools Used
- Microsoft Excel
- PivotTables
- Conditional Formatting
- Slicers
- Dashboard layout techniques
